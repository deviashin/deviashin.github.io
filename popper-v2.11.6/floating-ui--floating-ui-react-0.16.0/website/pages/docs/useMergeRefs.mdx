# useMergeRefs

Merges an array of refs, returning a memoized callback ref.

```js
import {useMergeRefs} from '@floating-ui/react';
```

<Notice type="version">Since `v0.15.0`</Notice>

Inside your component:

```js
const ref = useMergeRefs([ref1, ref2, ref3]);
```

This does not get passed to `useInteractions(){:js}`.
